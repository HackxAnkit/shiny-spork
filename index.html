About Cash Flow Minimizer

Cash Flow Minimizer is a smart expense splitting tool that helps you manage shared expenses among friends, roommates, or colleagues.

The app uses an optimized algorithm to minimize the number of transactions needed to settle all debts, making it easier and faster to split bills.

Features:

  • ✓ Add and manage expenses easily
  • ✓ Automatic settlement calculations
  • ✓ Visual balance sheet
  • ✓ Minimal transaction optimization
  • ✓ Dark/Light mode support
  • ✓ Multi-language support (English/Hindi)
